# NLP(Natural Language Processing) pipeline

- 목적: 단순히 하나의 개별 단어를 이해하는 것을 넘어, 해당 단어들의 문맥을 이해하려는 것

- e.g. 전체 문장 분류, 문장 내 단어 분류, 텍스트 컨텐츠 생성, 
질의에 대한 정답 추출, 입력 텍스트로부터 새로운 문장 생성
